## 写在前面

作为一个成年初学者，在断断续续学钢琴一年之后，我已经可以弹奏一些曲子。但与此同时，我常常感觉，比起一个懂音乐的人，我更像是一名打字员。能够把五线谱上的旋律变成两手交替的音符的确已经是一件了不起的事（对我自己而言），但除此之外，我还是想真正理解旋律为什么这样走，和弦为什么这样排。因此在这个独立日假期，我花了一些时间研究了一些简单的和声理论，并把它们整理成这篇笔记。

## 一、一个音，是有引力的

让我们从最简单的一个音开始。

坐到琴前，从 C 往上爬一遍音阶：C-D-E-F-G-A-B，然后停在最后那个 C 上。你会感到一种"到家了"的踏实。


```abc
X:1
T:回到 C——到家了
M:4/4
L:1/4
Q:1/4=72
K:C
C D E F | G A B c ||
```

现在再爬一遍，但这次**故意停在 B 上，不弹最后的 C**。那种悬着的、没说完的难受感立刻就来了——那个 B 在拼命想往上挪半步，回到 C。

```abc
X:1
T:停在 B——悬而未决
M:4/4
L:1/4
Q:1/4=72
K:C
C D E F | G A B z ||
```



这就是音乐里最小的一个单位：**张力，与它的解决**。而这里藏着第一条规律——**音和音之间是有"引力"的，引力的强弱，取决于它们离最近的"稳定音"有多近。**

在 C 大调里，C 是"家"（我们叫它主音，tonic）。围绕着它，七个音每一个都有自己的"性格"：

- **B**（导音，leading tone）离 C 只有半个音（semitone），贴得极近，像磁铁，所以它想回家的欲望最强烈。
- **F** 到 E 也是半音，所以 F 有一种轻轻想往下叹气、滑回 E 的倾向。
- 而 **D 和 A** 呢？它们上下最近的音都隔着一个全音（whole tone），距离远了，"磁力"就弱，于是它们听起来在游荡、不着急去哪，像句子间的过渡词。

**倾向的强弱 = 离最近稳定音的距离。** 这条规律其实我们的耳朵早就懂了，只是从没有人把它说出来。

## 二、为什么有的音"合"，有的音"打架"：泛音的秘密

接下来我们来看音与音之间的关系。通常我们最关心的问题是：为什么有的音，譬如 C 和 G 一起响，听起来那么和谐，而有些音凑在一起就刺耳？

答案藏在一个物理现象里：**泛音（overtones）**。

你按下钢琴上一个 C，这根弦不只按整根长度在震，它同时还按 1/2、1/3、1/4……分段地震动。每一个分段震得更快，发出的频率是基频的整数倍。于是一个"单音"里，其实同时藏着一串越来越弱的高音。假设你弹的 C 是 100 Hz（真实约 131 Hz，这里用整数好算）：

| 第几个泛音 | 频率 | 是什么音 | 和 C 的关系 |
|:---:|:---:|:---:|:---|
| 1（基频） | 100 | C | 它本身 |
| 2 | 200 | C | 高八度（octave） |
| 3 | 300 | **G** | 纯五度（perfect fifth） |
| 4 | 400 | C | 又一个八度 |
| 5 | 500 | **E** | 大三度（major third） |
| 6 | 600 | G | |

**两个音和不和谐，取决于它们的频率比能不能用简单的整数表示。** 比值越简单，两串泛音重叠得越多，听起来越"融"：

- 八度 C : C = **2 : 1**，最简单，几乎融成一个音。
- 纯五度 C : G = **3 : 2**，次简单——所以 G 又稳又和谐，像家里第二重要的支柱。

与之相反，比值越复杂，两串泛音重叠得越少，听起来越"刺耳"：

- 大二度 C : D = **9 : 8**，比值变复杂，泛音开始错开，能听出毛刺。
- 三全音 C : F♯ = **45 : 32**（数字记不住也没关系），泛音列几乎对着干，所以坐立难安。

把它们叠在一起听，差别会一下子分开。先这两组，几乎融成一个声部：

```abc
X:1
T:协和：泛音重叠
M:4/4
L:1/1
Q:1/4=48
K:C
"C : c (2:1)" [Cc] | "C : G (3:2)" [CG] ||
```

再听这两组，毛刺和悬疑感都来了：

```abc
X:1
T:不协和：泛音错开
M:4/4
L:1/1
Q:1/4=48
K:C
"C : D (9:8)" [CD] | "C : F#" [C^F] ||
```

因此，"和谐"不是玄学，就是泛音重叠的多寡，也就是音程的简单程度。

## 三、开心还是悲伤，压在一个音上

现在来解开那个最迷人的问题：为什么同一个和弦，大调听起来开心，小调听起来悲伤？

秘密全在那个 **第三个音（the third）** 上。对比两个比值：

- **大三度（major third）** C : E = **5 : 4** = 1.25 → 明亮、开心
- **小三度（minor third）** C : E♭ = **6 : 5** = 1.20 → 阴郁、悲伤

差别只有**一个半音**。但声学上的逻辑很清楚。

大三度的 E（5:4），**正好就藏在 C 的泛音列第 5 位**（见上节表格）——它本来就是 C 这根弦在轻声发出的音。所以当你弹 C-E-G 时，这
个 E 顺着、加强了 C 本来就有的泛音，整个和弦泛音高度吻合、互相支撑，于是听起来通透、明亮、"对"。

而小三度的 E♭（6:5）**不在** C 的低位泛音列里。它是个"外来户"，和 C 的泛音对不上，制造出一点摩擦和阴影，于是听起来阴郁。

C 大三和弦（C-E-G）和 C 小三和弦（C-E♭-G），**根音和五音完全一样，只换了中间那一个音**。

但色彩的天平就这么窄。我们用实际弹奏来感受一下。

先叠根音和三音，只比半音之差：

```abc
X:1
T:同一个 C，两个三音
M:4/4
L:1/2
Q:1/4=56
K:C
"C — E (5:4)" c, E | "C — E♭ (6:5)" c, _E ||
```

再把三音放回和弦里。注意 G 不变，只换中间那个音，听听看有什么变化：

```abc
X:1
T:三音变了，和弦就变味
M:4/4
L:1/4
Q:1/4=72
K:C
"C → E → G" C E G z |  "C → E♭ → G" C _E G z | 
"C" [CEG]4 | "Cm" [C_EG]4 ||
```


**大调的三音顺着泛音列，小调的三音逆着泛音列。** 明亮 = 顺应 C 弦本来就有的泛音，悲伤 = 故意偏开它。我们耳朵里的那点"忧郁"，很多时候就是这一个音和泛音列对不上产生的微妙不协和。

（补一句：情绪当然不只来自声学——文化、歌词、速度都会掺进来。学界对"小调为什么悲伤"也还有争论。但顺/逆泛音列这个框架，对这篇入门笔记来说已经够用了。）

## 四、最不安分的音程：三全音

往和弦走之前，还有一个音程值得单独拎出来——因为它会在后面几乎每一句"想回家"的和声里露面。

在 C 大调里，同时按下 **B 和 F**。那种坐立难安、像悬疑片前奏的感觉，和第二节里 C 叠 F♯ 其实是同一个东西：都是**三全音（tritone）**——相距 6 个半音，正好是八度的一半，频率比 **45 : 32**，泛音列几乎对着干。只不过在 C 大调里，它换了个写法：B 是第七音，F 是第四音，两个都是音阶里的"自己人"，凑在一起却最不安分。

它为什么这么"急"？因为第一节里两条单音的引力，在这里撞到了一起：

- **B** 离 C 只有半音，拼命想往上回家（导音）。
- **F** 离 E 也只有半音，轻轻想往下滑回去。

两股力同时拉满，却没有一个落脚点——所以它不能停，只能解决：

```abc
X:1
T:三全音：拆开听，再合起来听
M:4/4
L:1/4
Q:1/4=72
K:C
"B→C" B c z | "F→E" F E z2 |
"B + F" [BF]2 "C + E" [CE]2 ||
```

它为什么这么"急"？因为它内部装了两条拼命想解决的路线：**B 想上行到 C，F 想下行到 E**。这两股引力同时拉满，所以它必须解决。记住这个声音——它是后面一切"张力和弦"的核心零件。

## 五、和弦，以及"起承转合"

现在把单音搭成和弦。方法很简单：从一个音出发，**隔一个音取一个**（这叫"叠三度"）。C 跳过 D 取 E，跳过 F 取 G，于是得到 C-E-G，这就是 C 大三和弦（C major），我们的"家"。

```abc
X:1
T:叠三度：C 大三和弦
M:4/4
L:1/4
Q:1/4=72
K:C
C E G | [CEG]2 ||
```

围绕这个家，最重要的是它左右两个"五度的邻居"——还记得五度是泛音里最亲的关系（3:2）吗？我们就用它来选和弦：

- 往上一个五度：C → **G**。G 建出的和弦是**属和弦（dominant）**，代号 **V**。它是"张力方"，想解决回家。
- 往下一个五度：C ← **F**。F 建出的和弦是**下属和弦（subdominant）**，代号 **IV**。它是"离开方"，开阔而温暖。

> 为什么向上一个五度是”张力方“，而向下一个五度是”离开方“？
> 
> 我们来看看 V 级（G）和 IV 级（F）和弦跟主和弦（C）的血缘方向：
>
> * **向心运动（V → I）**：自然泛音列里，C 振动会生出 G。G 天然向 C 靠拢——弹 G 的时候，物理上的吸引力是拉你回家的。
> * **离心运动（I → IV）**：在泛音列里，是 F 振动才会生出 C。C 的身体里流着 F 的血。从 C 弹到 F，声学上像是回到了 C 的"根源"；从 C 的视角看，就是**走出了 C 的引力圈**。
>
> 从 C 到 G，和从 C 到 F，共享的音和变化的音也不一样：
>
> | 进行 | 共享的音 | 变化的音 | 听觉心理 |
> | --- | --- | --- | --- |
> | **C (C-E-G) → G (G-B-D)** | G | C→B，E→D | 保留了 G，又生出导音 B，**拉满弓弦、随时准备射回 C** |
> | **C (C-E-G) → F (F-A-C)** | C | E→F，G→A | E→F 和 G→A 都往上走，色彩进入**开阔的新领域** |

---



```text
    IV  ← 5度 →  I  ← 5度 →  V
    F            C          G
  (离开家)      (家)     (想回家的张力)
```

三个和弦分别听起来是这样的：

```abc
X:1
T:三大功能和弦
M:4/4
L:1/2
Q:1/4=60
K:C
"I (C)" [CEG] | "IV (F)" [FAC] | "V (G)" [GBd] ||
```

这三个和弦叫**三大功能和弦**：主（I）、下属（IV）、属（V）。它们能撑起几乎整个西方调性音乐，有两个原因：

**第一，它们三个加起来，正好把七个音全包了。** C(CEG) + F(FAC) + G(GBD) = C D E F G A B，一个不差。所以任何旋律音都能在这三个和弦里找到归宿。

**第二，它们造出了一个完整的情绪循环，也就是"起承转合"：**

- **I（家）**：稳定。起。
- **IV（离家）**：开阔，像舞台灯突然打开。承。
- **V / V7（张力顶点）**：它含有那个三全音，拼命想解决——是"话说了一半、观众屏息不敢鼓掌"的那一秒。转。
- **回到 I（到家）**：张力释放，掌声响起。合。

连起来就是完整的"起承转合"：

```abc
X:1
T:I → IV → V7 → I
M:4/4
L:1/2
Q:1/4=68
K:C
"C" [CEG] | "F" [FAC] | "G7" [GBdf] | "C" [CEG] ||
```

顺带说一下 **V** 和 **V7** 的区别：V（G-B-D）只有三个音；V7（G-B-D-**F**）多加一个 F，正好和 B 组成三全音。所以 **V7 的"回家"张力更急更强**——想要强烈的收束（尤其在句尾），就用 V7。

对比一下——两个和弦紧挨着听，差别更明显：

```abc
X:1
T:V vs V7
M:4/4
L:1/4
Q:1/4=72
K:C
"V (G)" [GBd]2 "V7 (G7)" [GBdf]2 ||
```

这就是最经典的"悬着，然后落地"：

```abc
X:1
T:V7 -> I：张力解决
M:4/4
L:1/4
Q:1/4=80
K:C
"G7" [GBdf] "C" [CEGc]3 ||
```

## 六、一个调，其实有七个和弦

我们只用了三个和弦，但一个调里天生就有**七个**。方法还是"叠三度"，只不过对**每一个音**都做一次：

```text
C → C E G      D → D F A      E → E G B
F → F A C      G → G B D      A → A C E      B → B D F
```

七个音，七个和弦。编号用根音在音阶里的位置：大写（I、IV、V）表示大和弦，小写（ii、iii、vi）表示小和弦，后面带 ° 的（vii°）表示减和弦。

```text
级数:   I     ii    iii   IV    V     vi    vii°
和弦:   C     Dm    Em    F     G     Am    Bdim
```

七个和弦依次听一遍：

```abc
X:1
T:C 大调七个和弦
M:4/4
L:1/4
Q:1/4=72
K:C
"I" [CEG]2 "ii" [DFA]2 | "iii" [EGB]2 "IV" [FAC]2 |
"V" [GBd]2 "vi" [Ace]2 | "vii°" [Bdf]2 ||
```

为什么有的是大和弦、有的是小和弦？**不用背，数半音就行**——看根音到三音之间隔了几个半音：

- **C → E**：C、C♯、D、D♯、E，**4 个半音** → 大三度 → 大和弦
- **D → F**：D、D♯、E、F，**3 个半音** → 小三度 → 小和弦

```abc
X:1
T:数半音：大 vs 小
M:4/4
L:1/4
Q:1/4=72
K:C
"C→E (4)" C E | "D→F (3)" D F ||
```

七个和弦全数一遍，在大调里永远得到同一个模板：

```text
I    ii    iii   IV   V    vi   vii°
大    小    小    大    大    小    减
```

**大、小、小、大、大、小、减。** 这就是为什么学会一个调，等于学会所有调。

而这也顺手解释了：**vi（Am）、ii（Dm）、iii（Em）之所以带来忧伤或柔和，正因为它们是小和弦**——它们的三音逆着泛音列。

三种大和弦（I、IV、V）还是三根柱子；三个小和弦（ii、iii、vi）是新的**色彩**——它们让你的调色盘从三种亮色，变成亮暗兼备，能画出情绪和层次了：

```abc
X:1
T:大和弦 vs 小和弦
M:4/4
L:1/4
Q:1/4=72
K:C
"I (C)" [CEG]2 "vi (Am)" [Ace]2 | "IV (F)" [FAC]2 "ii (Dm)" [DFA]2 ||
```

### 减和弦：不是一种情绪，是一个箭头

最后那个 **vii°（B-D-F）** 特殊在哪？大和弦、小和弦虽然明暗不同，但它们的外框（根音到五音）都是稳定的纯五度。而减和弦是两个小三度摞起来，根音到顶音变成了……**三全音**（B 到 F，第四节听过的那个）！所以它不是"开心还是难过"的问题，它是**极度不稳定、拼命想解决**。它本质上是 V7 的近亲（G7 去掉根音 G，剩下的正是 B-D-F）。它不是一种颜色，它是一个指向家的**箭头**：

```abc
X:1
T:vii° → I
M:4/4
L:1/4
Q:1/4=72
K:C
"vii°" [Bdf]2 "I" [CEG]2 ||
```

### 功能和情绪，是两条独立的轴

这里有个容易混的点，想通它就拿到了下一层的钥匙：**"功能"和"情绪"是两个独立的维度。**

- **功能**看的是"稳不稳、想不想解决"。
- **情绪**看的是"明还是暗"。

所以 vi（Am）可以是"稳定的家族成员"（功能上类似主和弦，因为它和 I 共享 C、E 两个音），**同时**又是"忧伤的"（情绪上是小和弦）。它是"忧伤版的家"——两个和弦叠在一起看，C 和 E 没动，只有三音从 G 换成了 A：

```abc
X:1
T:I vs vi：功能相近，色彩不同
M:4/4
L:1/4
Q:1/4=72
K:C
"I (C-E-G)" [CEG]2 "vi (A-C-E)" [Ace]2 ||
```

而减和弦是"不稳定"的——它们根本不在一条轴上。

七个和弦按功能分成三大家族：

| 家族 | 成员 | 功能 | 起承转合 |
|:---|:---|:---|:---:|
| 主（Tonic） | I, iii, vi | 家 / 稳定 / 放松 | 起 |
| 下属（Subdominant） | IV, ii | 离开 / 铺垫 | 承 |
| 属（Dominant） | V, vii° | 张力 / 必须解决 | 转 |

分家的依据很优雅：看它们和三根柱子**共享哪些音**。vi 和 I 共享 C、E，所以归主家族；ii 和 IV 共享 F、A，所以归下属家族——听一下，色彩不同，但"离开家"的功能很像：

```abc
X:1
T:IV vs ii：同族替换
M:4/4
L:1/4
Q:1/4=72
K:C
"IV (F-A-C)" [FAC]2 "ii (D-F-A)" [DFA]2 ||
```

vii° 内含三全音，和 V7 近亲，归属家族——和 V 一样，都想把音乐推回 I。

**换和弦，但不换功能**——这就是音乐能有无穷变化，却又始终"讲得通"的秘密。

## 七、和弦为什么这样接：两层引力

和弦不是随便串的。它的运动背后有**两层引力**同时在起作用。

### 第一层：功能引力

宏观的默认方向是：**主 → 下属 → 属 → 主**（起承转合）。为什么是这个方向而不能反过来？因为属功能（V）里的三全音一旦拉起来，就**只想往前冲回主**，不想倒退。所以"下属→属"是自然的累积，"属→主"是天经地义的解决，而"属→下属"听起来就像把拉满的弓又松掉，别扭。（摇滚里故意用 V→IV 制造"不落俗套"，但那是明知是逆流的效果。）

```abc
X:1
T:顺流 vs 逆流
M:4/4
L:1/4
Q:1/4=72
K:C
"G→C (顺)" [GBd]2 [CEG]2 | "G→F (逆)" [GBd]2 [FAC]2 ||
```

### 第二层：五度引力

微观的动力是：**一个和弦最想去的地方，是它下方五度的那个和弦。** 这还是泛音里"五度最亲（3:2）"的延伸——G 的下方五度正是 C，所以 V→I 是最强的解决，本质就是根音下行五度。

### 用两把尺子量两个套路

**套路一：I – V – vi – IV（C-G-Am-F）**

这是全世界流行歌里最泛滥的进行，有人叫它"Axis 进行"。它的灵魂在第二步到第三步：**G(V) 之后本该回到明亮的 I，它却滑到了忧伤的 vi**。因为 vi 和 I 共享两个音，这个替代成立、不突兀；可它又是小和弦，于是带来一个意外的忧伤转折。这个手法叫**假终止（deceptive cadence）**。它一次次让你以为要回明亮的家，却总落进忧伤的家——这就是它抓耳又略带惆怅的秘密。

真终止和假终止放在一起听，差别很清楚：

```abc
X:1
T:真终止 vs 假终止
M:4/4
L:1/4
Q:1/4=72
K:C
"G→C" [GBd]2 [CEG]2 | "G→Am" [GBd]2 [Ace]2 ||
```

这个 loop 的"耐听"来自它没有真正句号：

```abc
X:1
T:I - V - vi - IV
M:4/4
L:1/4
Q:1/4=92
K:C
"C" [CEG]2 "G" [GBd]2 | "Am" [Ace]2 "F" [FAc]2 :|
```

有意思的是它常常**结束在 IV 上**，听起来"没结束"。这不是缺陷：它本就是拿来**循环**的，收尾的 IV 正好用一个柔和的 IV→I（"阿门"式的变格进行）接回下一圈的开头。它牺牲了"结束感"，换来了"耐听的循环感"。

**套路二：4536251（卡农进行）**

翻译成级数是 IV-V-iii-vi-ii-V-I。它的魔力藏在后半段的根音走向：

```text
和弦:  iii   vi    ii    V     I
根音:  E  →  A  →  D  →  G  →  C
       ↘5th ↘5th  ↘5th  ↘5th
```

E→A→D→G→C，**每一步都是下行五度**！这是一条完美的"五度连锁"，像多米诺骨牌，一路势不可挡地滚回家。这就是为什么它听起来如此顺、如此有推进力、如此"必然"——它把音乐里最强的引力连续用了四五次。

```abc
X:1
T:4536251（卡农进行）
M:4/4
L:1/4
Q:1/4=76
K:C
"IV (F)" [FAC]2 "V (G)" [GBd]2 | "iii (Em)" [EGB]2 "vi (Am)" [Ace]2 |
"ii (Dm)" [DFA]2 "V (G)" [GBd]2 | "I (C)" [CEG]2 ||
```

**两个套路，两种哲学：** I-V-vi-IV 玩的是功能层的"意外"（假终止）；4536251 玩的是五度层的"连锁必然"。你以后听任何一首歌，都可以用这两把尺子去量。

### 顺带一提：流行 vs 古典

这两个套路也划出了两种和声哲学。**流行**追求的是"可以一直转下去的律动"（groove）——和声原地循环，靠旋律、编曲、力度去做变化。**古典**追求的是"结构性的收束"——乐段必须有明确的 V→I，像句子必须有句号，和声是发展的、叙事的，不是循环的。这个区别，正好通向最后一层。

## 八、离开一个调：串门与搬家

到此为止，我们所有和弦都来自**一个调**。但真正丰富的音乐会**离开这个房间**——去别处走一圈，制造惊喜、阴影、远行感，再回家。离开有两种方式，一轻一重。

### 轻的：借用和弦（串门）

**临时从别的调借一个和弦进来，用一下就还回去，调本身不变。**

最经典的一招，是从**同名小调**借。比如 C 大调里，把明亮的 IV（F 大三和弦）换成从 C 小调借来的 **iv（Fm，F-A♭-C）**。区别只在一个音：A 降成 A♭。但效果天差地别——弹 C→Fm→C，你会听到一股突如其来的、甜蜜又忧伤的揪心感。那个 A♭ 是"外来户"，像明亮天空里突然飘过一朵云。

这个 A♭ 一出现，明亮的家就被染上了阴影：

```abc
X:1
T:I - iv - I：借来的阴影
M:4/4
L:1/4
Q:1/4=64
K:C
"C" [CEG]2 "Fm" [F_Ac]2 | "C" [CEG]2 ||
```

它为什么有效？因为你的耳朵已经建立了 C 大调的预期，而 A♭ **违反了预期**——它是从"更悲伤的平行世界"借来的一抹阴影。音乐的感染力，很大程度就来自**建立预期，再巧妙打破它**。

**华语例子：周杰伦《菊花台》。** 副歌里用了一个小下属和弦 B♭m——它不属于原调 F 大调，而是从 F 小调借来的。你听副歌那句时那股凄美揪心，就是这个借来的小下属在起作用，和你在琴上弹 Fm→C 是同一个原理。（反过来也有：齐豫《橄榄树》是在小调里借了大调的下属，别有一番风味。）

### 重的：转调（搬家）

**音乐真正离开原调，在新的调里稳定下来，把"家"整个挪走。** 这是古典音乐叙事性的核心引擎。

怎么让耳朵相信"家搬了"？还是用那个最强工具——**V→I**。是什么让 C 成为家？是 G7→C 一次次告诉耳朵"C 是终点"。那么想搬到 G 大调，只要找到 G 的属和弦（D7），用 **D7→G** 去反复确立 G 就行。

```abc
X:1
T:转调：D7 → G
M:4/4
L:1/4
Q:1/4=72
K:C
"D7" [D^FAc]3 "G" [GBd] ||
```

> **想让某个音成为新的家，就用"它的 V7"去解决到它。**

它的轻量版叫**离调（secondary dominant）**——不彻底搬家，只是短暂地把某个和弦"临时当成家"强调一下，用完还回来。

**华语例子：周杰伦《千里之外》。** 副歌本来要走到下属和弦 G，作者在它前面临时搭了一组"属于 G 的 ii-V"（Am→D7），用你最熟的 V→I 把耳朵短暂骗到"G 才是家"，再滑回真正的调。这种短暂离开带来新鲜的色彩和推动力。这个 ii-V-I 离调是周杰伦的招牌手法，散落在他大量作品里。

```abc
X:1
T:离调：Am → D7 → G → C
M:4/4
L:1/4
Q:1/4=72
K:C
"Am" [Ace]2 "D7" [D^FAc]2 | "G" [GBd]2 "C" [CEG]2 ||
```

古典音乐则把转调用到极致，讲出一个完整的故事：

```text
在主调建立"家"
   → 转到属调，制造"离家在外"的张力
   → 去更远的调游历，冲突加深
   → 最终转回主调，"回家"，冲突解决
```

这就是它和"原地循环的流行 loop"最根本的区别：它用**调的移动**，模拟了离开、冒险、回归的完整叙事弧。

## 九、整张地图

我们从"一个音为什么想动"出发，一路走到了这里。回头看，它是这样长起来的：

```text
① 单音的引力      ┐
② 协和的物理      │  微观：声音本身的物理法则
③ 明暗的物理      │  （泛音、频率比、半音引力）
④ 三全音         ┘
       ↓
⑤ 和弦的功能      ┐
⑥ 七个和弦        │  中观：一个调的内部结构
   与三大家族      ┘  （功能、明暗、叠三度）
       ↓
⑦ 进行的语法      ┐  宏观：和弦如何运动
   （功能+五度引力）┘  （语法、套路、流行vs古典）
       ↓
⑧ 借用和弦        ┐  超越：如何离开一个调
⑨ 转调           ┘  （惊喜、远行、叙事）
```

**每一层都建立在下一层的物理之上，没有一处是背下来的规则——全都能从"频率比"和"引力"两样东西推导出来。**

而它正好覆盖了两种音乐：**古典**主要靠进行和转调讲叙事；**流行**主要靠家族替换和循环造律动，偶尔用借用和弦点睛。放到你熟悉的华语歌里，是这样落地的：

| 你学的概念 | 真实的用法 | 听感 |
|:---|:---|:---|
| 小和弦 = 忧伤色彩 | 1645 里的 vi（如《可爱女人》） | 压抑一下又转回明亮 |
| 五度连锁（4536251） | 卡农进行，大量抒情/中国风歌 | 绵延、顺、势不可挡地回家 |
| 借用小下属（iv） | 《菊花台》的 B♭m | 明亮里突然一抹凄美 |
| 离调 ii-V-I | 《千里之外》的 Am-D7-G | 短暂的新鲜感 + 推进力 |
